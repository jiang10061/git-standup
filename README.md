🎸 git-standup (瞎写的版本)

不可发送单个标点符号

😅 这是啥？

我作为一个业余编程爱好者，趁着周末，一边查百度一边瞎试，搞出了这个小东西。

它的主要作用是：帮我从电脑里那一堆乱七八糟的项目里，把昨天的 Git 提交记录翻出来，省得我一个一个点开看（我也记不住）。

🤔 它能干嘛？（我猜的，不一定对）

- 自动瞎翻：你告诉它代码大概在哪，它自己试着钻文件夹找，但不保证能找全。
- 强行说人话：我试着把提交记录前面的英文标签换成 Emoji 和中文，代码写得比较丑，可能匹配不上。比如：
   - 
"feat" -> 
"🌟 厉害的新功能" （希望是）
   - 
"fix" -> 
"🔧 终于修好了" （可能没修好）
   - 其他不认识的一律当作 
"📌" 处理。
- 防止电脑爆炸：我怕它把我电脑跑死，所以限制它只翻几层文件夹，再深的就找不到了。
- 随机关怀：如果是周末运行，它可能会劝你休息。代码是我拍的，触发逻辑可能很怪。

🐛 怎么用？（代码很菜，出问题别怪我）

1. 确保你电脑有 Python（版本不对我不管）。
2. 在终端（或命令行，叫啥都行）里输入：
python git-standup.py -d ~/你的代码路径
3. 祈祷别报错，然后看着它跑。跑完你把屏幕上的东西复制出来，能不能交差就看天意了。
